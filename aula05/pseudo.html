Pseudo Classes em links

Utilizadas para a formatação de algumas tags, como por exemplo a tag "a", por padrão o browser deixa o link azul e sublinhado.

  • a:link {atributos} <!--Como o link será formatado antes de ser clicado-->
  • a:hover {atributos} <!--Como o link será formatado ao passar o mouse sobre ele-->
  • a:visited {atributos} <!--Como o link será formatado depois de já ter sido clicado-->
  • a:active {atributos} <!--Como o link será formatado enquanto está sendo clicado-->
    •

Links formatados:
a:hover{
color:white;
background-color: blue;
}

a:visited{
color:red;
background-color: aqua;
text-transform: uppercase;
}

a:link{
text-decoration: none;
}

a:active{
border-bottom: 14px;
border-color: blueviolet;
}
