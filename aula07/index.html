Imagens

Há duas maneiras de se inserir imagens no documento HTML:



HTML
<img src="fonte da imagem" alt="texto disponível caso a imagem não carregue"> //alt também funciona para acessibilidade para deficientes visuais


CSS
background-image: url(caminho da imagem);

Obs: Se a imagem tiver função informativa, e necessitar de um maior destaque utilizar no html (img); caso seja uma imagem decorativa, com função estética, utilize o CSS;
