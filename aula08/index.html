Layout

Tags semânticas

<header>

utilizada para determinar qual conteúdo fará parte do cabeçalho da página.

<nav>

utilizada para determinar o setor onde fica a navegação do site, como o menu.

<main>

Utilizada para determinar o setor principal da página, que conterá as informações mais relevantes.

<footer>

Utilizada para determinar o setor de rodapé da página;

Tags de estrutura

<div>

(não possui utilidade semântica) faz uma espécie de divisória de conteúdo, onde é possível formatar cada uma especificamente.

<section>

utilizada para dividir o conteúdo em seções relevantes, ideal para uma melhor organização da página e do código.

<article>

utilizada para seções de conteúdo principal, com conteúdo específico, como o próprio nome diz, um artigo

<aside>

utilizada para conteúdo que seja à parte do conteúdo principal, como a propaganda por exemplo. O que está lá não faz parte do conteúdo principal.

Float

Controla a posição e o fluxo dos elementos na página. Tendo em vista que os elementos em bloco são caixas.

.DIV1
float: none;
background: rgb(252, 77, 214);
width: 200px;
height: 200px;

.DIV2
float: right;
background: rgb(47, 179, 255);
width: 200px;
height: 200px;

.DIV3
float: none;
background: greenyellow;
width: 100px;
height: 100px;

O bloco acima possui um width de 500px e height de 500px.
É como se fossem camadas do Photoshop.
OBS: assim que se coloca um float, o elemento deixa de fazer parte do elemento pai, que não mais o considera e gera um tamanho sem ele.
Elementos com Float se etendem e participam da mesma camada, se encaixam de maneira ordenada.
