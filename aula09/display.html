Display

Display Block

Elementos de bloco se comportam como se fossem caixas na página, ocupam 100% do espaço disponível e são dispostos um embaixo do outro, gera quebra de linha.
É possível aplicar formatações de box model.
<p>
<h1> ... <h6>
<ul>
<li>
<footer>
 etc.

Display Inline

Elementos de linha se comportam como se entrassem numa fila, ocupam seu espaço porém são dispostos em linha, ocupa o tamanho exato de seu conteúdo, um ao lado do outro.
<a>
<strong>
<em>
<i>
<span>
<img>
 etc.

Exemplo:

em{
display: inline-block; <!-- desta forma eu transformo um elemento inline em um elemento block, sem gerar quebra de linha -->
}
display: none; <!-- Desabilita da tela um elemento -- >

Box-sizing

Esta propriedade é utilizada para definir ao navegador que o elemento bloco em questão tem a sua largura definida pela propriedade width, o padding a ser aplicado não aumentará a sua largura e altura, mas se enquadrará no espaço definido.

p {
width: 400px;
background-color: yellow;
padding: 300px;
height: 400px;
color: rgb(48, 47, 47);
}

O código acima é um exemplo de como o padding não influencia quando se tem a propriedade box-sizing.
A caixa tem exatos 400px, mesmo com o padding de 100px;
