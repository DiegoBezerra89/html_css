Transform

Esta propriedade possibilita a modificações de algumas características do elemento, como sua posição no documento, rotacionar o elemento, modificar a escala do tamanho do elemento, e diversas outras funções.

div1
div2
    
    .div1{
        width: 150px;
        height: 150px;
        background: red;
        transform: translate(50px, 50px);/*Desta maneira o elemento será deslocado, 50px no eixo x e 50px no eixo y*/
    }

Porém, com esta propriedade desta maneira, o elemento não leva em consideração o tamanho do documento, ele se desloca a quantidade especificada não levando em consideração os outros elementos, por vezes ocupa os lugares dos mesmos.

E qual a diferença entre position e translate?
A principal diferença é em como o elemento se comporta quando é especificado %

div2



    
    .div3{
        width: 150px;
        height: 150px;
        background: red;
        transform: translate(50%, 50%);
    }

Na div acima ele se deslocou 50% de seu tamanho, no eixo x, assim como no eixo y.
Se fosse position:relative, o elemento se basearia no tamanho do elemento pai e não de si mesmo.

Skew

Este atributo modifica os eixos X e Y, como um free transform com perspectiva.

                                width: 150px;

                                transform: skew(10deg);
                            }

                                width: 150px;

                            }
