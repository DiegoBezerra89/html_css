Imagens arquivos externos e links

a

refere-se a links
Atributos:
  • href: refere-se a página de destino do link
  • target: refere-se ao modo de abertura do link

link

refere-se a links de arquivos externos
Atributos:
  • href: refere-se ao arquivo de destino do link
  • rel: refere-se ao tipo de arquivo que será linkado

img

refere-se a links de imagens
Atributos:
  • src (source): refere-se a origem da imagem
  • alt: texto alternativo caso a imagem não apareça

Script

refere-se a trechos de código baseados em scripts, normalemente código js
Atributos:
  • src: refere-se a origem do código (arquivo .js)
