Flexbox

Paradigma atual de formatação de layout das páginas web, funciona como um float, porém mais moderno e bem mais simples.
A grande mudança está relacionada a criar um conteiner com divs dentro, onde o container receberá a classe flexbox, com uma formatação definida, o próprio conteiner definirá a disposição dos itens inseridos nele.
