Listas ordenadas e não ordenadas

Podem haver dois tipos de listas ordenadas(ol) e não ordenadas(ul).
Dentro destas listas estão as "li" (list item).

  • nas ul não há ordem numérica.
  • nas ol há ordem numérica.

ul - unordered list

ol - ordered list

Links

Links servem para ir para uma página ou local na página usando a tag "a".
