Position - relative e absolute

Esta diretriz faz referência à posição do objeto na tela:
div{
position: static; <!-- Padrão, onde os objetos são postos em ordem, um abaixo do outro, inline ou block.-->
}

No caso de relative e absolute, as diretrizes trabalham com os parâmetros left e right, pois elas tem referência a elas mesmas ou a outro objeto.

.div1{
position: relative;
left: 30px;
top: 30px;
background: yellowgreen; }
.div2{
position: static;
left: 30px;
top: 30px;
background: rgb(4, 196, 132);
}
.div3{
position: static;
left: 30px;
top: 30px;
background: rgb(18, 129, 92);
}

Observe na imagem acima que foi aplicado a position: relative, a posição do bloco é RELATIVA à ela mesma, deslocando-se a quantidade descrita nas propriedades.

position: absolute

O position: absolute, se relaciona sempre com o primeiro elemento pai posicionado na página.
Se o elemento pai estiver com position: static; , o elemento com o position absolute se referencia ao documento HTML.
Se o elemento pai estiver com position: relative , o elemento com position absolute se referencia ao elemento pai.

.div4{
position: absolute;
right: 30px;
top: 30px;
background: yellowgreen; width: 150px; }
.div5{
position: static;
left: 30px;
top: 30px;
background: rgb(4, 196, 132);
}
.div6{
position: static;
left: 30px;
top: 30px;
background: rgb(18, 129, 92);
}
