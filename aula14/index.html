Semântica

É necessário ao usuário saber o que o campo deseja receber, qual o tipo de informação é necessária inserir nos inputs.
Há diversas ferramentas semânticas que facilitam as entradas de dados.

Label

<label for="txtBusca"> Faça uma busca </label>
<input type="text" name="q" id="txtBusca">
 Desta forma "Faça uma busca" se referencia semânticamente com o input em questao. Como a tag evidencia, é uma espécie de etiqueta.

Fieldset

<form action="pagina.php" method="POST">
<fieldset>
<legend>Login</legend>
<label for="log">User:</label>
<input type="text" name="text" id="log">
<label for="pass">Password:</label>
<input type="password" name="pass" id="pass">
</fieldset>
<input type="submit" value="Login">
</form>

Atributos de inputs

autofocus:
Coloca o cursor do mouse no elemento em questão.
checked:
Faz com qe a opção fique marcada.
required:
A informação inserida no input é obrigatória
multiple:
Funciona em conjunto com o select, torna as opções múltiplas ao segurar control
placeholder:
Texto que existe antes do usuário utilizar uma caixa de texto, geralmente contém dicas de preenchimento dos campos.
