Inserindo Mídia

Áudio

    
    <audio src="media/mozart_trompa.mp3" controls> /*é necessária a diretiva controls, para mostrar os controles de áudio.

Obs: a formatação do controle de áudio muda conforme o browser

- os Browsers também podem ter peculiaridades quanto ao reconhecimento de extensões de arquivos de áudio, por isso se faz necessário o uso da tag source 

        
        <audio controls autoplay> /*é necessária a diretiva controls, para mostrar os controles de áudio.*/    /*autoplay, assim que a página é carregada o som é reproduzido (alguns browsers não tocam)*/
        <source src="media/mozart_trompa.mp3" type="audio/mp3"></source>
        <source src="media/mozart_trompa.ogg" type="audio/ogg"></source>/*type indica ao browser qual é o tipo do arquivo a ser reproduzido*/
        <p>Seu Browser não suporta este arquivo</p>/*sáo vai aparecer quando o browser não conseguir executar o arquivo*/

iFrame

Os iframes são janelas externas ao site
São formatáveis por CSS. 

        
            <iframe src="../aula01/index.html">

OBS: alguns sites não permitem que você incorpore o conteúdo em seu site, por medida de segurança.
Iframes possuem alguns atributos(checar documentação).
