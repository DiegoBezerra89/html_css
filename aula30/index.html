Animações com CSS

Transition

Para falarmos de animação é necessário dar atenção a propriedade transition, que deixa a animação mais fluida:

  • transition-property: diz qual a propriedade a sofrer a transição.
  • transition-duration: tempo em que a transição ocorre.
  • transition-delay: tempo de atraso da transição.
  • transition-timing-function: maneira que a transição se comporta.
  • transition: property duration timing-funciton delay;

Múltiplas transições:

        
    transition: property1 duration1 timing-function1 delay1,
                property2 duration2 timing-function2 delay2;
    
    transition-property:            property1, property2;
    transition-duartion:            duartion1, duration2;
    transition-timing-function:     func1, func2;
    transition-delay:               delay1, delay2;

Evento gatilho

Para a transição entrar em funcionamento é necessário um evento para acioná-la: hover, :active, focus, etc. Sempre é necessária uma interação do usuário.

Animation e Keyframes

Keyframes cria uma animação, são quadros intermediários que fazem que a animação aconteça. Transições mais complexas, não dependem de interação do usuário, e possui maior controle sobre a animação criada.

Timeline


Neste caso há três estados de animaçao, cada um com 1 segundo de duração, sempre determinar os keyframes em porcentagem, o tempo da duração de cada etapa do keyframe depende do tempo total da animação.

Keyframes ou quadros-chave

Pontos cruciais para que a animação aconteça. Eles que ditam como serão calculados os frames intermediários, feitos pelo computador automaticamente.
Somente valores numéricos podem ser animados: 10px, .5, 58%, 10em, #cf0, red.
Valores como: auto, arial, hidden, entre outros não são numéricos.

Transform e opacity

São as propriedades mais performáticas para animação.

will-change

Se preciso utilize esta propriedade para o browser se preparar previamente para o uso da animação. OBS: SÓ UTILIZE QUANDO NECESSÁRIO!

Keyframes

    
        @keyframes animaCor{                @keyframes animaCor{
            from{                               0% {
                color: red;                        color: red;         
            }                                   }
            to{                                 100% {        
                color: blue;                        color: blue;
            }                                   }                
        }                                   }

Ambos fazem a mesma coisa. Já que todos modificam o elemento completamente em duas etapas, início e fim.

        
            @keyframes animaCor{       
                0%{                        
                    color: red;               
                }                                   
                50%{                   
                    color: blue;
                }  
                100%{
                    color: orange;
                }                               
            }

Este keyframe é acionado em três etapas.

Aplicando o keyframe

        
            .elemento{
                animation: animaCor 1s; /*shorthand*/
                /* maneira completa
                  animation-name: animaCor;
                  animation-duration: 1s;*/
            }
