CSS | Cascading Style Sheets

Definição

Arquivos responsáveis pela formatação e estilização dos elementos HTML, utiliza de classes e Ids para se referir a determinadas seções e elementos da página.

Sintaxe: seletor { propriedade: valor;}
Seletor h1 { color: valor;}

Declarações

  • Inline: junto com o código html se coloca o atributo style na tag que deseja formatar, este método não é correto.
  • Interno: No arquivo html vc coloca a tag style e determina a CSS no mesmo código, o que torna o código maior e faz uso desnecessário de espaço
  • Externo: Referência a arquivo externo, o que torna a manutenção mais fácil e o código fica mais limpo. (Maneira Correta!)

Seletores de Elemento

Pode-se referir-se ao elemento diretamente:
p {color: green;}
Desta forma vc formata todos os elementos "p" da página.

Há também maneiras de personalizar a formatação, por meio da criação de classes e Ids para os elementos:

por exemplo: <p class="paragrafo01"> Desta meneira vc cria uma especificação para os paragrafos que contem a classe "paragrafo01"
No CSS:
.paragrafo01 {color: blue}

<p id="paragrafo01"> Desta maneira vc cria uma especificação para um id, que é único por página.
No CSS:
#paragrafo01 {color: blue}

Propriedades

Font:
  • font-family: especifica a família da fonte em questão, Sans-serif, serif, etc.
  • font-size: especifica o tamanho da fonte
  • font-style: especifica o estilo da fonte
  • font-variant: modifica a meneira de escrita
  • font-weight: largura da fonte
text:
  • text-allign: alinhamento do texto em relação ao elemento em que está inserido
  • text-decoration: sublinhado, negrito, itálico, etc.
  • text-indent: indentação do texto
  • text-transform: maiúsculo, minúsculo, etc.
  • color: cor da fonte.

Modos de cor:

RGB: Red, green, blue
Descrição: #FFF000 (seis digitos hexadecimais que representam as cores);
